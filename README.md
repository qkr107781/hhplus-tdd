## ✅ 커밋 메세지 컨벤션
| 타입 | 설명 |
| --- | --- |
| feat | 새로운 기능 |
| fix | 버그 수정 |
| docs | 문서 관련 수정 |
| style | 코드 스타일 변경 (코드 포매팅, 세미콜론 누락 등)기능 수정이 없는 경우 |
| refactor | 리팩토링 |
| chore | 설정, 빌드, 잡일 등 |
| test | 테스트 코드 추가 |
| remove | 파일을 삭제만 한 경우 |
| rename | 파일 혹은 폴더명을 수정만 한 경우 |
| perf | 성능 개선 |
| security | 보안 관련 |
